# WiFiKey2 インストールガイド

> **[English version is here](INSTALL.md)**

このガイドは、開発環境なしで WiFiKey2 を使い始めるための手順を説明します。

---

## 必要なもの

| アイテム | 説明 |
|----------|------|
| Windows PC (Windows 10/11) | サーバーアプリを動かすPC |
| M5Atom Lite × 1 | パドル側（クライアント） |
| USB-Cケーブル | M5Atom LiteのUSB接続（書き込み・電源供給） |
| CWパドル / ストレートキー | M5Atom Liteに接続するキー（3.5mmジャック経由） |
| フォトカプラ回路 | キー出力と無線機の接続（PCサーバー構成ではRS-232Cアダプタ） |
| WiFiルーター | クライアントとサーバーが接続できるネットワーク |

---

## 構成パターン

### パターンA：PCサーバー構成（推奨）

```
[パドル] → [M5Atom Lite] →(WiFi)→ [PC wifikey-server] → [無線機 CATポート/KEYポート]
```

PCにインストールした **wifikey-server** がリグ制御とキーイングを担当します。

---

## ステップ 1：サーバーアプリのインストール（パターンAのみ）

### 1-1. インストーラーのダウンロード

[GitHub Releases](https://github.com/jl1nie/wifikey2/releases) から最新版の
`WiFiKey2_x.x.x_x64-setup.exe` をダウンロードします。

### 1-2. インストール

1. `WiFiKey2_x.x.x_x64-setup.exe` を実行
2. 言語選択（日本語 / English）
3. 画面の指示に従ってインストール
4. 管理者権限は不要です（ユーザーフォルダにインストールされます）

### 1-3. 初回起動

スタートメニューまたはデスクトップの **WiFiKey2** を起動します。

---

## ステップ 2：サーバーの設定（パターンAのみ）

### 2-1. 設定画面を開く

アプリ右上の **⚙ 設定ボタン** をクリックします。

### 2-2. 必須項目を設定

| 設定項目 | 説明 | 例 |
|----------|------|-----|
| **サーバー名** | クライアントが接続する識別子 | `JA1XXX/keyer` |
| **接続パスワード** | クライアントとの認証パスワード | 任意の文字列 |
| **キーイングポート** | DTR/RTSでキーイングするCOMポート | `COM3` |
| **RTS使用** | DTRの代わりにRTSを使う場合にON | 無線機に合わせて |

> **サーバー名** はクライアント側の設定と完全一致させてください。

### 2-3. リグ制御（オプション）

Yaesu FT-891 などのCATコントロールをする場合：

| 設定項目 | 説明 |
|----------|------|
| **リグコントロールポート** | CATケーブルのCOMポート |
| **Luaスクリプト** | `yaesu_ft891.lua` など |

### 2-4. 設定を保存して閉じる

「保存」をクリックして設定を反映します。

---

## ステップ 3：ESP32へのファームウェア書き込み

### 3-1. M5Burnerのインストール

[M5Stack公式サイト](https://docs.m5stack.com/en/download) から **M5Burner** をダウンロードしてインストールします。

### 3-2. M5BurnerでWifikey2を検索・書き込み

1. M5Atom Lite を USB-Cケーブルでパソコンに接続
2. M5Burner を起動
3. 左上の検索ボックスに **`Wifikey2`** と入力して検索
4. 表示された **Wifikey2** をクリック
5. **COM ポート** を選択（不明な場合はデバイスマネージャーで確認）
6. **「Burn」** ボタンをクリックして書き込み開始
7. "Done" が表示されたら完了

---

## ステップ 4：クライアントの初期設定

書き込み直後のM5Atom Liteはプロファイルが未設定なので、自動的に **APモード** で起動します。

### 4-1. APモードの確認

- M5Atom Lite のLEDが **青色に点滅** していればAPモードです
- 青点滅しない場合は、電源を入れ直してください

### 4-2. WiFiに接続

スマートフォンまたはPCのWiFi設定を開き、以下のネットワークに接続します：

| 項目 | 値 |
|------|-----|
| **SSID** | `WifiKey-XXXXXX`（XXXXXXはデバイス固有の英数字） |
| **パスワード** | `wifikey2` |

### 4-3. 設定画面を開く

ブラウザで `http://192.168.71.1` にアクセスします。

WiFiKey2 の設定画面が表示されます。

### 4-4. プロファイルを追加

画面下部の **「Add New Profile」** フォームに入力します：

| 項目 | 説明 | 例 |
|------|------|-----|
| **WiFi SSID** | 接続するWiFiの名前 | `MyHomeWiFi` |
| **WiFi Password** | WiFiのパスワード | `wifipassword` |
| **Server Name** | wifikey-serverで設定したサーバー名 | `JA1XXX/keyer` |
| **Server Password** | wifikey-serverで設定した接続パスワード | （設定したパスワード） |
| **Tethering** | スマホのテザリングで使う場合にON | 通常はOFF |

入力後、**「Add Profile」** をクリックします。

### 4-5. 再起動

**「Save & Restart」** をクリックします。M5Atom Lite が再起動し、設定したWiFiへ接続を開始します。

---

## ステップ 5：接続確認

1. wifikey-server を起動
2. M5Atom Lite に電源を入れる
3. LEDが **黄色** → サーバーを探索中
4. LEDが **消灯** → サーバーへの接続成功
5. wifikey-server のタイトルバーが **赤色** になれば接続完了

---

## 基本的な使い方

### LED の意味（クライアント側 M5Atom Lite）

| 色 | 状態 |
|----|------|
| 赤点灯（起動時一瞬） | 起動中 |
| 青点滅 | APモード（設定待ち） / WiFi再接続中 |
| 黄点灯 | サーバー探索中 |
| 消灯 | 接続完了・待機中 |
| 白点灯 | キーON（送信中） |
| 赤点灯 | ATU起動中 |

### ボタン操作

| 操作 | タイミング | 動作 |
|------|-----------|------|
| 短押し | 運用中 | ATUスタート |
| 5秒長押し | 電源投入直後 | APモードに移行（設定変更） |

### プロファイルの変更・追加

設定を変更したい場合は、**電源投入直後にボタンを5秒長押し** してAPモードに入り、
`http://192.168.71.1` から設定を変更します。

### 複数のWiFiプロファイル

複数のプロファイルを登録しておくと、起動時に周囲のWiFiを自動スキャンして
最初に見つかったSSIDへ接続します（自宅WiFiとモバイルテザリングを両方登録するなど）。

## トラブルシューティング

### LEDが青点滅にならない（APモードにならない）

- NVSにすでにプロファイルが保存されている可能性があります
- 電源投入直後にボタンを**5秒長押し**してAPモードに入ってください

### `WifiKey-XXXXXX` が見つからない

- M5Atom Lite の電源が入っているか確認してください
- 数秒待ってからWiFiリストを更新してみてください

### `http://192.168.71.1` が開けない

- スマートフォンの場合、接続したWiFiがインターネットに繋がらないため「別のネットワークを使用」などのポップアップが出ることがあります。「このまま接続」を選択してください
- PCの場合、ブラウザのアドレスバーに直接 `http://192.168.71.1` と入力してください（`https://` ではなく `http://`）

### 接続できない（LEDが黄点灯のまま）

- サーバー名とパスワードが一致しているか確認してください（大文字・小文字を区別します）
- wifikey-server（PCサーバー）が起動しているか確認してください
- クライアントとサーバーが同じWiFiに繋がっているか確認してください
- ファイアウォールがUDP通信をブロックしていないか確認してください

### WiFiには繋がるがサーバーに接続できない（WAN経由）

- テザリングを使用している場合は、プロファイルの **Tethering** をONにしてください
- MQTTブローカー（`test.mosquitto.org`）への接続が必要です。プロキシ環境の場合は接続できないことがあります

## バージョンアップ方法

### wifikey-server（PC）

新しい `WiFiKey2_x.x.x_x64-setup.exe` をダウンロードして実行するだけです。
設定は上書きされません。

### M5Atom Lite ファームウェア

M5Burnerの **Wifikey2** から最新版を書き込むだけです。
**NVSのプロファイルは保持されます**（設定のやり直し不要）。
